coven

Summon AI agents. Orchestrate with precision. A unified platform for running, connecting, and managing AI agents—locally or in the cloud.

Get Started View Source
~/.local/bin/coven
# Start a local gateway
$ coven serve
✓ Gateway running on localhost:7890
# Connect an agent
$ coven agent run --name oracle
Capabilities

Everything you need to orchestrate AI agents

⬡

Local Gateway

Run a super-trusted local gateway for development. No cloud required. Full control over your agents.

coven serve
⌘

Unified CLI

One command-line interface to manage agents, gateways, packs, and swarms. Scriptable and composable.

coven --help
◈

Terminal UI

Interactive TUI for real-time agent conversations. Stream responses, approve tools, manage sessions.

coven chat
⎔

Swarm Orchestration

Coordinate multiple agents across workspaces. Supervisor mode, parallel execution, shared context.

coven swarm
⊞

Tool Packs

Extend agent capabilities with modular tool packs. MCP bridge, productivity tools, custom integrations.

coven pack run
⟁

Multi-Frontend

Connect via CLI, TUI, Matrix, Slack, or Telegram. Same agents, any interface you prefer.

coven link
